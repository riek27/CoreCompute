Installation

Install the HyperAstraAI Python SDK using pip:

pip install hyperastraai

For development versions, you can install directly from GitHub:

pip install git+https://github.com/hyperastraai/sdk-python.git
Note: Python 3.8 or higher is required. We recommend using a virtual environment.

Authentication

To use the HyperAstraAI platform, you need to authenticate using your API key. You can get your API key from the dashboard after signing up.

Setting up your API Key

Set your API key as an environment variable:

export HYPERASTRA_API_KEY='your-api-key-here'

Or set it directly in your code:

import hyperastraai as haa

client = haa.Client(api_key='your-api-key-here')
Warning: Never commit your API key to version control. Use environment variables for production applications.

Your First Job

Here's a simple example of submitting a TensorFlow training job:

import hyperastraai as haa

# Initialize client
client = haa.Client()

# Define job configuration
job_config = {
"framework": "tensorflow",
"model_type": "image_classification",
"gpu_tier": "mid",
"dataset": "cifar10",
"epochs": 10,
"batch_size": 32
}

# Submit job
job = client.jobs.submit(job_config)
print(f"Job submitted with ID: {job.id}")

# Monitor progress
job.wait_for_completion()
print("Job completed!")

For PyTorch users, the process is similar:

import hyperastraai as haa

client = haa.Client()

job_config = {
"framework": "pytorch",
"model_type": "text_classification",
"gpu_tier": "entry",
"model": "bert-base-uncased",
"dataset": "imdb",
"epochs": 3
}

job = client.jobs.submit(job_config)
job.wait_for_completion()

HyperAstraAI provides a RESTful API and Python SDK for programmatic access to all platform features.

/v1/jobs/{job_id}/logs
Retrieve job execution logs
DELETE
/v1/jobs/{job_id}
Cancel a running job

Jobs API

The Jobs API allows you to submit, monitor, and manage AI training and inference jobs.

Submit a Job

import hyperastraai as haa

client = haa.Client()

# Submit a job
job = client.jobs.submit(
framework="tensorflow",
model_type="image_classification",
gpu_tier="mid",
credits=50,
script="training_script.py",
requirements="requirements.txt"
)

Monitor Job Progress

# Check job status
status = job.status
print(f"Job status: {status}")

# Wait for completion
job.wait_for_completion()

# Get results
results = job.get_results()
model = results.download_model()
